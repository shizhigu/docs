## Overview

SkoutLab connects to your data wherever it lives. Once connected, our AI continuously learns your data structure and business context.

## Supported Sources

### File Uploads

| Format | Description |
|--------|-------------|
| **Excel** (.xlsx, .xls) | Full support including multiple sheets |
| **CSV** | Standard comma-separated values |
| **Parquet** | Columnar format for large datasets |

### Databases

| Database | Status |
|----------|--------|
| **PostgreSQL** | Supported |
| **MySQL** | Supported |
| **MotherDuck** | Supported |
| **BigQuery** | Coming soon |
| **Snowflake** | Coming soon |

## Multi-Source Analysis

SkoutLab excels at analyzing data across multiple sources:

SkoutLab will automatically join and analyze data from both sources.
